盒盖不关机

2021-05-29 :: kasusa
#ubuntu  #命令行 

1、编辑 logind.conf 文件，命令如下

  sudo vim /etc/systemd/logind.conf

2、把所有相关 HandleLidSwitch 的值都改成 ignore

3、执行让修改的内容即时生效，输入命令：

systemctl restart systemd-logind

(实时生效命令可能不能用，我这边ubuntu20.04使用了之后不停卡登录界面)
